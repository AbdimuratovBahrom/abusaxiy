## Что это такое?

Это веб-приложение помогает продавцам, электрикам и администраторам ТЦ «Абу Сахий» быстро находить:

- По какому **ЩР / ЯРВ / ШО** подключён конкретный магазин
- Через какое **ТП / ВРУ / АВР** идёт питание
- В каком **блоке** и **ряду** находится магазин
- Специальные объекты (туалеты, офисы, АСКУЭ, кафе и т.д.)

Просто выбираешь блок → ряд → магазин → увидишь **полный путь** от ТП до магазина.

## Как пользоваться

1. Выберите **блок** (1-блок, 2-блок, 3-блок, Гипермаркет, Специфические объекты)
2. Выберите **ряд** (A, B, J, 1, 2, 5 и т.д.)
3. Выберите **магазин** из выпадающего списка
4. Сразу увидите **полный путь** электроснабжения

Или используйте **быстрый поиск** (вводите например `3-1-50`, `г-269`, `туалет`)
